## 数据融合/新-基础控件

新-基础控件

2.1 表格

将维度列的值配置在度量列区域

先决条件（案例需求）：

配置方法：

for(var i = 0; i < data.length; i++){

var row = data[i];

row["附加度量"] = row["地区"];

}

return false;

return cell;

滚动联动

用于表格与其他控件联动时，当其他控件选定一个数值时，表格会自动滚动到该条记录。

首先将滚动联动打开。当省份选择浙江时，表格自动到该行记录。

配置表格的导出

1.为表格配置名称，基础->名称

2.添加“图标按钮”控件(方形元素），配置：行为->点击事件->数据导出->选择表格名称

3.保存报表

4.预览报表，点击导出按钮,导出的表格数据以excel形式导出，如图所示。

1. 如果表格配置了【表格宽度】，【宽度类型】就不能设置为固定

2. 如果【宽度类型】设置了固定，就不要配置【表格宽度】

3. 如果设置了行转列，最好是将【宽度类型】设置为固定

2.2 单值图

用于通过文本显示指标，通过多个单值图组成指标拼盘。

【注】此处会用到”设置”>>报表页>>列数 功能

点击图表分类“基础控件”，点击“单值图”，选择模拟器中组件“配置”，为单值图配置数据源。

配置项有数据、排序、过滤、函数、预警、配置、选择项、标题、基本、边框、浮动、行为、联动。

下面详细介绍配置

配置

风格：主值字体风格

尺寸：主值字体大小

对齐：主值对齐方式

颜色：主值颜色

左右间隔：相对原始对齐方式的位置，偏移位置，正值往右偏移，负值往左偏移

小数位：配置数值小数位数

千分位：设置数值千分位

值类型：（A）实际值 显示原始值、（%）百分比 对原始值*100%

在数据源中选中两个度量，才会显示出副值，数据源中。

2. 副值配置：

显示：默认关闭

位置：右侧（主值右侧）、底部（和副标题并齐）

风格：字体风格

尺寸：字体大小

颜色：字体颜色，也可以通过“预警”功能，给字体设置对应值域颜色

左右间距：“位置”是右侧时，相对于主值的偏移距离，“位置”是底部时，相对于副标题的偏移距离。正值向右偏移，负值向左偏移。

主值脚本原型：

function(度量名){

return 脚本内容;

}

示例：销售额/10000

2.4 控件容器

注：页面容器是把页面嵌入，里面的控件不能编辑

控件容器是直接在当前页面将控件加入容器，可以编辑，可以方便对控件进行分组， 对分组进行标题、边框、背景的配置

1、在页面添加控件容器，及需要填入的控件

2、设置控件容器，点击容器，单击内容控件输入框，即可 选择控件，选择即可添加控件

3、单击控件容器里面的控件设置，即可添加数值进行设置‘

’

4、容器列数对应报表页设置的列数

2.5 图片

引用静态图片，一般用于展现公司logo，或者首页宣传内容

1. 点击图表分类“基础控件”，点击“URL”，选择模拟器中组件“配置”

2. 在“图片”中，在附件中，选择静态图片。上传图片到附件中

2.6 多维图片

零售行业或者服装行业，通常在切换产品时，需要展示产品的示例图，此时可以用多维图片展示。

1. 点击图表分类“基础控件”，点击“多维图片”，选择模拟器中组件“配置”，选择数据源

多维图片数据源结构如下图，含有图片URL维度

2. 数据源维度顺序配置，最下面的维度是组件展现维度，其他为隐藏维度，开启“选择项”后才显示

3. 切换维度，显示对应的图片

2.8 URL

加载指定链接地址的页面，可以是报表链接地址，或者浏览器的链接地址，如下图：

2.9 图标按钮

接合“行为”功能，给图标按钮配置跳转等行为，作为报表内容入口按钮。如下图：

1. 点击图表分类“基础控件”，点击“图标按钮”，选择模拟器中组件“配置”

2. 在“配置”中，配置图标按钮需要的样式；在“基本”中配置背景色

配置项有：配置、基本、边框、浮动、行为。

下面详细介绍配置

文本：按钮上文本内容

风格：字体加粗、倾斜、下划线

尺寸：字体大小

颜色：字体颜色

上下间距：使用“上图标”时，图标和文本之间的间距

左右间距：使用“左图标”时，图标和文本之间的间距

配置左图标icon、尺寸和颜色

配置下图标icon、尺寸和颜色

2.10 容器

容器中可以选择展现该报表内指定的报表页页面。

1. 点击图表分类“基础控件”，点击“容器”，选择模拟器中组件“配置”

2. 在“容器”中配置展现的报表页

2.11 系统时间

系统时间，显示当前系统时间

点击图表分类“基础控件”，点击“系统时间”，选择模拟器中组件“配置”，配置系统时间。

配置项有：文本、标题、基本、边框、浮动、行为。

下面详细介绍文本。

文本

样式：标准文本和自定义，自定义支持脚本编辑

字体：支持多种字体样式

尺寸：配置字体大小

对齐：支持左对齐、居中、右对齐

字体颜色：配置字体颜色

字体风格：支持加粗、倾斜、下划线

颜色：阴影颜色

宽度：阴影字体宽度

水平、垂直偏移：相对于原字体，阴影偏移方向和距离
