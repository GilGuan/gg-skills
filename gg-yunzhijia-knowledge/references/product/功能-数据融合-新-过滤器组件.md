## 数据融合/新-过滤器组件

新-过滤器组件

过滤器控件种类有以下几种，分别为：文本标签过滤器、下拉框过滤器、轮盘过滤器、输入框过滤器、日期过滤器、导航控件、范围过滤器、搜索过滤器、多选过滤器、自定义过滤器。

注意

过滤器选择了的维度，同一报表页面中的其他控件不能再选择该维度。

3.1 过滤器的基本配置

选择过滤器控件后，对控件进行配置，如图选择组件上的配置。

配置项

选择数据集中需要应用的数据表作为过滤器控件的数据表。

设置维度作为不同过滤器的过滤元素，并设置对应的度量。

（如：维度为客户所在省，度量为销售额、均价）

包括排序、过滤、函数、图形、选择项、基本、边框、浮动、联动项的设置。

3.2 过滤器控件的使用效果与场景

1. 文本标签过滤器

选择同维度中不同标签，根据选中名称对报表中显示数据进行过滤。

2. 下拉过滤器

在下拉框中分别选择筛选条件，可共同对报表数据进行过滤，可以实现多选、设置多维度。

3. 轮盘过滤器

点击过滤面板并滑动以筛选条件（维度）。可以设置多个维度。

4. 输入框过滤器

数据

图形

表达式：支持=、≦、≧、LIKE、IN、NOT IN，相当于sql中where条件，根据需要选择表达式

圆角：输入款圆角

高度：输入框高度

左、右、上边距：输入框相对组件外边框距离

参考下拉选择器“标签和标签别名”

5. 日历过滤器

注意

日历过滤器不需要数据绑定，但需要在“图形”配置项的“过滤 --> 列名” 中输入 要过滤的列名，如下图：

【日历】

默认值：无（不配置）、今天、本月第一天、本年第一天、数据表、脚本

格式：YYYY-MM-DD ，自定义以你的日历数据为准；如数据源日历为YYYY/MM/DD、无分隔符YYYYMMDD等

偏移：-1即昨天，0为当天，1为明天

【过滤】

列名：直接输入要过滤列名（即数据表的字段名）

表达式：≤、=、≥

【标签】

显示：开启、关闭显示标签

别名：标签的（开始、截止、选择日期等）提示名称

位置：左侧、顶部可选

字体尺寸：标签的字体尺寸

【默认值脚本】

var date = moment().add(-10,'days').toDate();

return date; //需要返回一个javascript的Date对象

日历控件的效果

在弹窗中输入要展示的日期。

8.搜索过滤器

在查询框中输入过滤元素。

（1）支持模糊搜索。

（2）搜索后勾选项目进行筛选，也可在查询历史中勾选。

（3）如果将部门名称、客户名称、省份信息同时设置为过滤器维度，则都可在查询框中搜索并同时过滤。

9.多选过滤器

10.自定义过滤器

需在选择项中自行编辑脚本使用。

脚本样例

今天

var format = "YYYY-MM-DD";

var columnName = "column1";

var date = moment().format(format);

return columnName+" = '"+date+"'";

最近3天

var format = "YYYY-MM-DD";

var columnName = "column1";

var date = moment().add(-4, 'days').format(format);

return columnName+" > '"+date+"'";

返回值可以是字符串或者对象

var values = ["北京","上海"];

return {

column:"地区", //字段名

exp: "in", //表达式，可以使用   <, >, <>, in 等SQL表达式

value: values.join(",") //

}

操作类似于导航控件，点击控件选项触发脚本设定的事件。
