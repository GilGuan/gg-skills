## 云之家AI/知识智能体（AI+企业知识）

知识智能体（AI+企业知识）

什么是知识智能体

💡企业 AI 落地的基础“智能体”，企业私域知识贯穿 AI 知识问答、任务执行的全过程，安全（权限可控）、可靠（混合检索精确召回）、自迭代（知识自动同步到AI）。

知识库

知识中心同步

知识中心的文档同步到小K的知识库中，小K即可在回答用户问题时参考企业知识。

同步范围：所选目录内的文章内容

(含附件，支持的附件格式为.txt、.md、.csv、.xls、.xlsx、.pdf、.pptx、.docx)

用户在小K进行问答时，仅会参考当前用户在知识中心有权限查看的文档内容

知识中心的开放 API 丰富，可通过 API 将知识从第三方系统同步至知识中心

术语词典

管理员可以在后台维护术语词典，例如企业特有的缩写、专业术语、产品名称、项目代号等，并提供详细解释。在用户问题预处理阶段和检索阶段，系统会使用该词典进行查询扩展和实体识别，显著提升对企业“黑话”的理解能力，提高召回率和准确性。
