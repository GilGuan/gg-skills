## 智能助理/推送通知

推送通知

推送通知的使用场景

可以设置报表推送的通知内容，进行个性化预警推送。

一、 触发器规则

1、两个触发函数

函数

描述

alarm.add()

将当前数据行内容加到变量dataRow中，等待数据表的所有行都扫描完后再触发推送，可在通知内容中使用该变量适用场景，适用于下面的场景1；

alarm.send()

立即发送推送通知

适用场景：根据数据权限分别发送通知给不同的用户，比如店长，适用于下面的场景2。

2、应用场景

两种应用场景的用法

1、向指定用户发送列表型的信息

例如：向总经理发送一个各大区的信息列表，该场景下，不支持数据权限，必须配置接收者

使用 alarm.add()

​

2、向有数据权限的用户发送单条信息

例如：向各个区域经理发送自己的指标情况，该场景下，支持数据权限，不需要配置接收者，有权限的用户自动收到信息

使用 alarm.send()

​

三、预警通知脚本配置

1、使用变量

使用变量的格式：{{字段名}}

例如：

访问Map对象格式： 变量名["字段名"] ， 例如遍历dataRow时：

{{item["销售额"]}}

​

2、变量的计算

可以直接使用计算符号，例如：

//直接计算字段变量

{{ 销售额/10000 }} 万元

{{ ( (销售额-去年销售额) / 去年销售额 ) * 100 }}

​

//遍历dataRow时，计算字段列，必须对字段使用括号

{{ ( ( (item["销售额"])-(item["去年销售额"])) / (item["去年销售额"]) ) * 100 }}
