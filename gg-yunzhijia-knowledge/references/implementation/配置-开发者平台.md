## 开发者平台

开发者平台

开发者平台

开发者平台是云之家统一应用开发平台，企业成员可以通过开发者平台进行应用开发，扩展和提升企业办公一站式办公能力

应用管理列表

应用管理是开发者所开发的应用列表，在列表中可以看到应用上下架、审核等所有状态

新建应用

点击右上角新建应用按钮，填写基本信息后，点击确定完成应用的初步创建

应用详情

应用详情包含了一个应用的所有信息

详细信息

该名称作为应用唯一名称，将作为应用标题对最终用户显示（如在应用中心、门户等入口）

ID与密钥

AppID

应用唯一的 ID 标识，调用接口时使用

App Secret

应用的密钥，用于换取 Access Token 等鉴权服务

Sign Key

用于调用第三方待办快捷同意服务

功能开发

功能开发选项卡包含了应用的核心内容，也是应用的主体

小程序

小程序是运行在云之家的媲美 Native 性能的应用！

通过云之家专为小程序打造的运行环境和开发工具，仅使用前端 HTML、JS、CSS 技术，即可开发出媲美 Native 性能的应用，无论是加载速度还是使用体验，都远超以往H5，极大的提高企业业务运转效率

点击查看小程序开发指南

激活小程序后，需要将已经打包完成的小程序包上传至开发者平台

上传完成后可以看到小程序的基本信息，以及预览小程序

轻应用

轻应用是运行在云之家客户端或Web门户中的网页程序。

平台级轻应用是云之家提供的通用应用，包括智能审批、智能考勤、时间助手、智能会议等。它们可以开箱即用，也可以与企业已有业务系统进行对接。

企业自建应用是企业针对自身业务创建的，它与平台级应用一样，也可以在云之家客户端中运行并利用云之家客户端能力和用户身份。

点击查看轻应用开发指南

激活轻应用后，需要将已经开发完成的 H5 URL 填入轻应用信息中，并对轻应用打开方式等进行配置

成员管理

开发者平台支持多开发者协同开发，在应用的成员管理中可以添加其他开发者，被邀请的开发者都拥有该应用的开发权限

点击右上角可以添加企业内其他成员为协同开发者

同时成员管理支持多角色，可以按需为成员分配权限

应用提审

当应用开发完成后，下一步即可对应用进行提审，点击应用详情右上角提审按钮

填写提审信息后，点击确定；

确定后该应用将会被提交到应用中心处，由应用管理员进行审核上架

点击查看应用中心产品手册

QA

Q：是否可以同时开启小程序与轻应用

A：是支持的，当用户在 PC、移动端打开应用时，会进行判断，如果当前应用有小程序版本，则优先打开小程序，反之则打开轻应用版本；常见用法如仅开发了移动端小程序，那么可以移动端使用小程序，PC 端使用轻应用
